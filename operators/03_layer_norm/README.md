# Layer Normalization（层归一化）

## 为什么需要归一化？

在深度神经网络中，每一层的输入分布会随着训练不断变化（这被称为 **内部协变量偏移**）。如果某一层的输入值范围很大（比如有的特征是 0.01，有的是 1000），网络训练会变得很不稳定。

**归一化的核心思想**：把数据"拉"到一个统一的范围内，让每个特征都有相似的尺度。

## 从均值和方差说起

### 均值（Mean）

均值就是"平均值"——把所有数加起来除以个数：

$$
\mu = \frac{1}{n} \sum_{i=1}^{n} x_i
$$

**例子**：有一组数 $[2, 4, 6, 8]$

$$
\mu = \frac{2 + 4 + 6 + 8}{4} = \frac{20}{4} = 5
$$

### 方差（Variance）

方差衡量数据"离均值有多远"——先算每个数与均值的差，再求差的平方的平均：

$$
\sigma^2 = \frac{1}{n} \sum_{i=1}^{n} (x_i - \mu)^2
$$

**例子**：还是 $[2, 4, 6, 8]$，均值为 $5$

$$
\sigma^2 = \frac{(2-5)^2 + (4-5)^2 + (6-5)^2 + (8-5)^2}{4} = \frac{9 + 1 + 1 + 9}{4} = \frac{20}{4} = 5
$$

### 标准差（Standard Deviation）

标准差就是方差的平方根：

$$
\sigma = \sqrt{\sigma^2} = \sqrt{5} \approx 2.236
$$

## Layer Normalization 公式

LayerNorm 对**每个样本**（每一行）的**所有特征**（最后一个维度）进行归一化：

$$
y = \frac{x - \mu}{\sqrt{\sigma^2 + \epsilon}} \cdot \gamma + \beta
$$

其中：
- $x$ 是输入向量
- $\mu$ 是该向量的均值（沿最后一个维度）
- $\sigma^2$ 是该向量的方差（沿最后一个维度）
- $\epsilon$ 是一个很小的常数（如 $10^{-6}$），防止除以零
- $\gamma$（weight）是可学习的**缩放参数**，与特征维度等长
- $\beta$（bias）是可学习的**偏移参数**，与特征维度等长

### 分步理解

1. **中心化**：$x - \mu$，让数据以 0 为中心
2. **缩放**：除以 $\sqrt{\sigma^2 + \epsilon}$，让数据的标准差约为 1
3. **仿射变换**：乘以 $\gamma$ 再加 $\beta$，让网络"学习"最合适的分布

## 手算示例

假设输入向量 $x = [1.0, 2.0, 3.0, 4.0]$，权重 $\gamma = [1.0, 1.0, 1.0, 1.0]$，偏置 $\beta = [0.0, 0.0, 0.0, 0.0]$，$\epsilon = 10^{-6}$。

**第一步：求均值**

$$
\mu = \frac{1 + 2 + 3 + 4}{4} = 2.5
$$

**第二步：求方差**

$$
\sigma^2 = \frac{(1-2.5)^2 + (2-2.5)^2 + (3-2.5)^2 + (4-2.5)^2}{4} = \frac{2.25 + 0.25 + 0.25 + 2.25}{4} = 1.25
$$

**第三步：归一化**

$$
\hat{x}_i = \frac{x_i - 2.5}{\sqrt{1.25 + 10^{-6}}} \approx \frac{x_i - 2.5}{1.11803}
$$

$$
\hat{x} = \left[\frac{-1.5}{1.11803},\ \frac{-0.5}{1.11803},\ \frac{0.5}{1.11803},\ \frac{1.5}{1.11803}\right] \approx [-1.3416,\ -0.4472,\ 0.4472,\ 1.3416]
$$

**第四步：仿射变换**（因为 $\gamma=1, \beta=0$，结果不变）

$$
y = [-1.3416,\ -0.4472,\ 0.4472,\ 1.3416]
$$

### numpy 验证

```python
import numpy as np

x = np.array([1.0, 2.0, 3.0, 4.0])
eps = 1e-6

mean = x.mean()           # 2.5
var = x.var()              # 1.25  (注意：numpy 默认用 N 而不是 N-1)
y = (x - mean) / np.sqrt(var + eps)

print(y)  # [-1.3416  -0.4472   0.4472   1.3416]
```

## 在 Qwen2-VL 中的应用

在 Qwen2-VL 的 **Vision Encoder**（视觉编码器）中，每个 Transformer Block 都包含 LayerNorm。例如第一个 block 的 `norm1`：

- 输入形状：`(14308, 1280)` —— 14308 个 token，每个 token 有 1280 维特征
- LayerNorm 对每个 token 的 1280 维特征独立做归一化
- 可学习参数：`weight` 形状 `(1280,)`，`bias` 形状 `(1280,)`

这意味着 LayerNorm 会执行 14308 次，每次处理一个长度为 1280 的向量。

## 关键要点

| 特性 | 说明 |
|------|------|
| 归一化维度 | 最后一个维度（特征维度） |
| 可学习参数 | weight ($\gamma$) 和 bias ($\beta$) |
| $\epsilon$ | 通常为 $10^{-6}$，防止除零 |
| 计算步骤 | 减均值 → 除标准差 → 缩放 → 偏移 |
